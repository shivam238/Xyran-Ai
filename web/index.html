// get started

Install in seconds.

One command. Any Linux distro. Also available on macOS and Windows.

bash 
# Clone the repo
git clone https://github.com/shivam238/Xyran-Ai.git
cd Xyran-Ai

# Fedora dependencies
sudo dnf install -y python3-dbus python3-gobject xdotool

# Setup & run
python3 -m venv venv && source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
python xyran.py

// four steps

1
Clone the repo from GitHub and enter the project directory.
2
Install system deps for your distro (DBus, PyGObject, xdotool).
3
Configure API keys — Groq & Gemini keys in your .env file.
4
Run python xyran.py — on first start, the embedding model (~90MB) auto-downloads. Then you're live.
